Extends WordPress WP REST API (V2) with new endpoints pointing to WooCommerce page functions (is_shop, is_cart, is_checkout, is_account_page).

== Description ==

This plugin extends the [WordPress JSON REST API](https://wordpress.org/plugins/rest-api/) with new endpoints for WooCommerce (is_shop, is_cart, is_checkout, is_account_page) page functions

== Installation ==

This plugin requires having [WP API](https://wordpress.org/plugins/rest-api/) and [WooCommerce](https://wordpress.org/plugins/woocommerce/) installed and activated or it won't be of any use.

Install the plugin as you would with any WordPress plugin in your `wp-content/plugins/` directory or equivalent.

Once installed, activate WP API WooCommerce endpoints from WordPress plugins dashboard page and you're ready to go, WP API will respond with new endpoints.
